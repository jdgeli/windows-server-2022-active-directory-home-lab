# Server Installation

## Objective

Install Windows Server 2022 and prepare it for Active Directory deployment.

## Environment

- Hypervisor: VMware Workstation
- OS: Windows Server 2022 Standard
- Network: NAT

## Steps

1. Download Windows Server 2022 ISO.
2. Create a new virtual machine.
3. Allocate 6 GB RAM and 2 CPU cores.
4. Install Windows Server.
5. Configure static IP address.
6. Rename the server.
7. Install the latest updates.

## Result

The server was successfully installed and prepared for Active Directory deployment.
